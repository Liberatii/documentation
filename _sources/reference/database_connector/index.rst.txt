.. _database_connector:

Liberati database connector
===========================

The Liberatii Database Connector is a replacement database connector for the Application's
existing Oracle Database Connector. Implementations are available to replace the following
types of database connector:

- :ref:`jdbc`
- :ref:`odbc`
- :ref:`oci`

See the :ref:`Technical Assessment <assessment>` for information and guides to help
determine the type of connector required, how it may be replaced and further information
for troubleshooting.
